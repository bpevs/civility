Color is Opt-in

All other color comes from explicit variants or application-specific styling. This ensures:

  • Complete themeability - change --body and --background to theme everything
  • Accessible contrast - always inherits proper contrast from text
  • Predictable behavior - no surprise colors
  • Easy customization - override text color for sections to theme them

Framework maintains neutral-first approach: color is always intentional, never accidental.
